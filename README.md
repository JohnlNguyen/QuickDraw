# Quick Draw GANs



## Getting Started



### Prerequisites

Requires Python >= 3.6

### Installing


```
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt

python -m ipykernel install --user --name venv --display-name "YOUR DISPLAY NAME"
```
